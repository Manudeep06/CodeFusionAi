## Technology Stack

### Frontend

* **React.js** – Builds a dynamic and responsive user interface.
* **Tailwind CSS** – Provides modern and efficient UI styling.
* **Monaco Editor** – Powers the in-browser code editor (same editor used in VS Code).
* **Xterm.js** – Renders a terminal console for CLI command interaction.
* **WebContainer API** – Runs a WebAssembly-based Node.js environment in the browser to run and preview applications.
* **Socket.IO Client** – Enables real-time collaboration and synchronization between users.
* **Firebase Authentication** – Handles secure user login and authentication using Google Sign-In.

### Backend

* **Node.js** – Provides a scalable server-side runtime environment.
* **Express.js** – Simplifies API development and backend routing.
* **Socket.IO** – Facilitates real-time bidirectional communication between connected users.
* **Mongoose** – Connects and models data structures for MongoDB.

### Database

* **MongoDB Atlas** – Cloud-based NoSQL database for storing user, project, and collaboration data.

### Artificial Intelligence

* **Gemini API** – Provides AI-powered code review, suggestions, bug detection, and code optimization recommendations.

### Code Execution Engine

* **WebContainers (Client-Side)** – Executes JavaScript/Node.js files and web templates (React, Vue, Svelte) inside a browser WASM sandbox.
* **Local Process Execution (Server-Side)** – Runs Python, C++, and Java code securely using server-side host compilers and interpreters via `child_process.exec`.
